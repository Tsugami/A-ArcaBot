# A Arca Bot

Discord bot for a server of friends who play monster hunter

## Getting Started

> Note: Node.js 16.12+ is required!

To run this project, install it locally using yarn:

```bash
yarn install
```

Copy .env.example file and change the bot token for your bot:

```bash
cp .env.example .env
```

Run project:

```bash
yarn dev
```
